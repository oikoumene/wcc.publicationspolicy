wcc.publicationspolicy Installation
-----------------------------------

To install wcc.publicationspolicy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.publicationspolicy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.publicationspolicy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
